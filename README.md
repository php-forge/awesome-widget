## Installation

The preferred way to install this extension is through [composer](https://getcomposer.org/download/).

Either run

```shell
composer require --prefer-dist php-forge/awesome-widget:"^0.1"
```

or add

```json
"php-forge/awesome-widget": "^0.1"
```
